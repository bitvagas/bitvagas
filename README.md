##Tools

* NodeJS + Express
* Sequelize + Postgre SQL
* Passport
* AngularJS
* HarpJS
* BrowserSync


##Usage

####Install the npm dependencies:

`$ npm install`

####Install the bower dependencies:

`$ bower install`

#### Configure your database

`# create database bitvagas_dev;`

#### Run fixtures:

`$ gulp fixtures`

####Run with:

`$ gulp`

Open your browser at `http://localhost:9000`
